# DefenseIQ

DefenseIQ-tech-learning platform is a full-stack cybersecurity/tech related learning platform that combines spaced repetition, AI-assisted study, progress analytics, and hands-on labs for technical learners.

## Included
- Next.js frontend
- Express + TypeScript backend
- Deck marketplace
- Progress analytics
- Spaced repetition engine
- AI flashcard generator scaffold
- AI tutor scaffold
- Code runner scaffold
- Cybersecurity labs scaffold
- PostgreSQL schema
- Docker Compose
- Worker scaffold
- Sample seed data
