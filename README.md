# Tech Learning Platform

A monorepo for a technical learning platform focused on cybersecurity, software engineering, and IT study workflows.

## Included
- Next.js frontend
- Express + TypeScript backend
- Deck marketplace
- Progress analytics
- Spaced repetition engine
- AI flashcard generator scaffold
- AI tutor scaffold
- Code runner scaffold
- Cybersecurity labs scaffold
- PostgreSQL schema
- Docker Compose
- Worker scaffold
- Sample seed data

### Start dependencies
